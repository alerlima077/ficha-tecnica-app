# 🍕 Ficha Técnica App - Júlio's

Sistema de Ficha Técnica para gestão de custos em Pizzaria, Restaurante e Padaria.

## 🚀 Funcionalidades

- **Dashboard** com 3 setores (Pizzaria, Restaurante, Padaria)
- **Cadastro de Insumos** com cálculos automáticos de custo por porção
- **Cadastro de Pré-Preparos** (massa base, molhos) que viram insumos automaticamente
- **Ficha Técnica Simplificada** (8 campos) para produtos finais
- **Cardápio Gerencial** com análise completa de precificação:
  - CMV, Margem de Contribuição, % Custo (com cores)
  - CET (Custo Efetivo Total), Margem Real
  - PV Ideal e Status de Precificação
- **PWA** (Progressive Web App) - pode ser instalada no celular
- **Exportação para CSV**

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript (ES6+)
- Firebase Firestore (Banco de dados)
- PWA (Manifest + Service Worker)
